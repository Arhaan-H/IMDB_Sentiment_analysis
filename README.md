# 🎬 IMDB Sentiment Analysis

A full end-to-end ML sentiment analysis pipeline trained on the **IMDB 50K Movie Reviews** dataset, served via a modern dark-themed Streamlit app.

## 📸 App Preview

> Live Predictor tab opens by default — type any movie review and get an instant POSITIVE/NEGATIVE prediction with confidence score and top 5 influential words.

## 📂 Dataset

Download the IMDB 50K Movie Reviews dataset and place it in the project root as `IMDB_Dataset.csv`:

👉 **[Download IMDB Dataset from Kaggle](<ADD_KAGGLE_LINK_HERE>)**

> The dataset file is **not included** in this repository due to its size (~66 MB). You must download it separately before running `train.py`.

## ✨ Features

- **3 ML models trained and compared**: Logistic Regression, Multinomial Naive Bayes, Linear SVM
- **TF-IDF vectorization**: 10,000 features, unigrams + bigrams
- **Negation-aware preprocessing** — words like "not", "no", "didn't" are preserved so "not good" ≠ "good"
- **Best model auto-selected** by F1-score and saved as a pickle
- **4-tab Streamlit UI**:
  - 🔮 **Live Predictor** — instant predictions with confidence score + top 5 influential words
  - 📊 **Data Overview** — dataset stats, class distribution, review length histogram, sample reviews
  - 🧹 **Preprocessing** — before/after cleaning comparison, top 20 words per class
  - 🤖 **Model Results** — comparison table, confusion matrix, interactive ROC curves with AUC

## 🚀 Setup & Run

### 1. Clone the repo
```bash
git clone https://github.com/Arhaan-H/IMDB_Sentiment_analysis.git
cd IMDB_Sentiment_analysis
```

### 2. Install dependencies
```bash
pip install -r requirements.txt
```

### 3. Download NLTK data
```bash
python -c "import nltk; nltk.download('stopwords'); nltk.download('wordnet')"
```

### 4. Place the dataset
Download `IMDB_Dataset.csv` from the Kaggle link above and place it in the project root folder.

### 5. Run training *(creates 3 pickle files — only needs to run once)*
```bash
python train.py
```

This saves:
- `vectorizer.pkl` — fitted TF-IDF vectorizer
- `best_model.pkl` — best performing trained model
- `model_results.pkl` — all metrics, chart data, sample reviews

### 6. Launch the app
```bash
streamlit run app.py
```

Open [http://localhost:8501](http://localhost:8501) in your browser.

## 📝 Notes

- `app.py` **never trains** — it only loads pre-computed pickle files for instant startup
- Re-run `train.py` only if you change the dataset or preprocessing logic
- Both `IMDB_Dataset.csv` and `IMDB Dataset.csv` filenames are supported
